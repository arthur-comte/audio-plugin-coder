# Style Guide v1

## Color Palette
- **Primary (Wet Channel):** `#FF1493` (Deep Pink)
- **Accent (Looper Channel / Global):** `#98FF98` (Mint Green)
- **Background:** `#1E1E24` (Dark Slate)
- **Panel / Section Backgrounds:** `#2A2A35` (Slightly lighter slate for depth)
- **Text:** `#F0F0F0` (Off-White)
- **Knob Bases:** `#111111` (Dark metallic)

## Typography
- **Primary Font:** Sans-serif (e.g., Inter, Roboto, or system default clean font).
  - Headers: 16px, Bold, Uppercase, tracking (letter-spacing)
  - Labels: 12px, Regular, Uppercase
  - Values: 11px, Light

## Layout & Spacing
- **Margins:** 20px around the main window.
- **Padding:** 15px inside sections (Wet, Looper, Global).
- **Corner Radius:**
  - Window/External: 10px
  - Sections/Group Boxes: 6px
  - Buttons: 4px

## Visual Style
- **Theme:** Modern Boutique Hardware (Minimalist but vibrant).
- **Controls:** Dark metallic knobs with brightly colored indicator rings (Deep Pink for Wet, Mint Green for Looper and Global).
- **Looper Waveform:** Drawn using a 2px stroke in `#98FF98` (Mint Green) over a very dark background `#111111` to emulate an LCD screen embedded in the pedal.
